To reproduce the analysis and results, follow these steps:

### 1. Clone the Repository

```bash
git clone https://github.com/Reverent2005/ML-project-classification.git
```
```bash
cd ML-project-classification
```

### 2. Create and Activate a Virtual Environment

It is highly recommended to use a virtual environment to manage project dependencies.

```bash
# Create a virtual environment named 'myvenv'
python3 -m venv myvenv
```
```bash
# Activate the environment
source myvenv/bin/activate
```
(Your terminal prompt should now show (myvenv))

### 3. Install Required Libraries

Install all the necessary packages using the requirements.txt file.

```Bash
# Install all libraries
pip install -r requirements.txt
```

### 4.Run the Jupyter Notebooks

You can now run the notebooks. The easiest way is to open the project in VS Code.
You can now explore and all the models and notebooks.
When prompted, select the kernel from your virtual environment (e.g., Python 3.x.x ('myvenv')).
